# Projeto-Store-Game
A GameStore atende a um público amplo (de 14 a 60 anos), oferecendo desde PCs gamers de alta performance até notebooks para trabalho e smartphones.  O design prioriza a clareza, utilizando uma paleta de cores focada em tons escuros e azuis, tipografia *Sans Serif* e navegação fluida, sem o excesso de informações visuais.
