# 🎮 GameStore - Front-End de E-commerce

Um projeto prático de desenvolvimento Front-End para uma loja de eletrônicos voltada ao universo gamer e de tecnologia. O layout foi construído com base em um briefing rigoroso de UX/UI, resultando em uma interface minimalista, responsiva e com foco na experiência do usuário.

## 📋 Sobre o Projeto

O objetivo deste projeto foi transformar um levantamento de requisitos e um protótipo gráfico em uma interface funcional e dinâmica. A GameStore atende a um público amplo (de 14 a 60 anos), oferecendo desde PCs gamers de alta performance até notebooks para trabalho e smartphones.

O design prioriza a clareza, utilizando uma paleta de cores focada em tons escuros e azuis, tipografia *Sans Serif* e navegação fluida, sem o excesso de informações visuais.

## ✨ Funcionalidades

* **Página Inicial (Home):** Navegação intuitiva com menu responsivo (Navbar).
* **Banner Dinâmico:** Carrossel de imagens destacando lançamentos e equipamentos de alta performance.
* **Vitrine de Produtos:** Cards responsivos divididos por categorias de público (Gamers, Estudantes, Profissionais).
* **Galeria Interativa:** Exibição de produtos com efeitos de transição (*hover*).
* **Sistema de Notificações:** Alertas dinâmicos para avisos de estoque e novidades.
* **Página de Autenticação:** Tela de login isolada com validação de formulário em tempo real.
* **Carrinho de Compras:** Simulação de adição de itens com atualização de contador em tela.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização customizada e Media Queries para responsividade.
* **Bootstrap 5:** Framework CSS para o sistema de grid, componentes (Navbar, Cards, Carrossel) e utilitários.
* **JavaScript (Vanilla):** Lógica fundamental e manipulação de eventos (ex: Carrinho de compras).
* **jQuery:** Facilitação da manipulação do DOM, validação de formulários e animações (FadeIn, temporizadores).
